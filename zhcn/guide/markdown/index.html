设置文本的格式
本文档是PenCake的Markdown用户指南。从PenCake 3.10开始支持文本格式。
PenCake使用Markdown来设置文本格式。Markdown是一种可以使用符号在纯文本中设置格式的语言。使用Markdown可以快速轻松地设置文本格式。
PenCake对常用的Markdown语法进行了一些修改，以更灵活和直观地设置格式。因此，下面的Markdown语法只与PenCake兼容。
文本样式
粗体
这是**粗体文本**。 This is **bold text**.
Plain Text
这是粗体文本。
This is bold text.
斜体
这是//斜体文本//。 This is //italicized text//.
Plain Text
这是斜体文本。
This is italicized text.
粗体和斜体
这是//**粗体和斜体**//文本。 This is //**bold and italicized**// text.
Plain Text
这是粗体和斜体文本。
This is bold and italicized text.
下划线
这是__下划线的文本__。 This is __underlined text__.
Plain Text
这是下划线的文本。
This is underlined text.
删除线
~~地球是平的。~~现在我们都知道地球是圆的。 ~~The world is flat.~~ We now know that the world is round.
Plain Text
地球是平的。现在我们都知道地球是圆的。
The world is flat. We now know that the world is round.
突出显示
这是==突出显示的==文本。 We now know that ==the world is round==.
Plain Text
这是突出显示的文本。
We now know that the world is round.
样式组合
==//**__~~这样使用也__~~**//==是可以的。 **==__符号的顺序==**__无关紧要。 ==其他格式__之间可以==交叉设置__。
Plain Text
这样使用也是可以的。
符号的顺序无关紧要。
其他格式之间可以交叉设置。
❌  无效的使用示例
文本样式不能紧挨着符号内侧有空格。
This is ** bold text**. ❌ This is //italicized text //. ❌ Really //** very **// important text ❌
