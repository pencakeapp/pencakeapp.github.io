設定文字格式
本文件是PenCake的Markdown使用手冊。從PenCake 3.10開始支援文字格式。
PenCake使用Markdown來設定文字格式。Markdown是一種可以使用符號在純文字中設定格式的語言。使用Markdown可以快速輕鬆地設定文字格式。
PenCake對常用的Markdown語法進行了一些修改，以更靈活和直觀地設定格式。因此，下面的Markdown語法只與PenCake兼容。
文字樣式
粗體
這是**粗體文字**。 This is **bold text**.
Plain Text
這是粗體文字。
This is bold text.
斜體
這是//斜體文字//。 This is //italicized text//.
Plain Text
這是斜體文字。
This is italicized text.
粗體和斜體
這是//**粗體和斜體**//文字。 This is //**bold and italicized**// text.
Plain Text
這是粗體和斜體文字。
This is bold and italicized text.
底線
這是__底線的文字__。 This is __underlined text__.
Plain Text
這是底線的文字。
This is underlined text.
刪除線
~~地球是平的。~~現在我們都知道地球是圓的。 ~~The world is flat.~~ We now know that the world is round.
Plain Text
地球是平的。現在我們都知道地球是圓的。
The world is flat. We now know that the world is round.
螢光筆
這是==用螢光筆突出顯示的==文字。 We now know that ==the world is round==.
Plain Text
這是用螢光筆突出顯示的文字。
We now know that the world is round.
樣式組合
==//**__~~這樣使用也__~~**//==是可以的。 **==__符號的順序==**__無關緊要。 ==其他格式__之間可以==交叉設置__。
Plain Text
這樣使用也是可以的。
符號的順序無關緊要。
其他格式之間可以交叉設置。
❌  無效的使用示例
文字樣式不能緊挨著符號內側有空格。
This is ** bold text**. ❌ This is //italicized text //. ❌ Really //** very **// important text ❌
Plain Text

一級標題
二級標題
