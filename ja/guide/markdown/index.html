書式を設定する
この文書はPenCakeのマークダウンユーザーマニュアルです。テキスト書式はPenCake 3.10以降でサポートされます。
PenCakeはマークダウン(Markdown)を使用してテキスト書式を設定します。マークダウンは、記号を使用してプレーンテキストに書式を設定できる言語です。マークダウンを使用すると、すばやく簡単に書式を設定できます。
PenCakeでは、より柔軟で直感的な書式編集のために一般的に使用されているマークダウン文法を一部変形して適用しました。したがって、以下のマークダウン文法はPenCakeでのみ互換できます。
テキストスタイル
太字
これは**太字**です。 This is **bold text**.
Plain Text
これは太字です。
This is bold text.
斜体
これは//斜体文字//です。 This is //italicized text//.
Plain Text
これは斜体文字です。
This is italicized text.
太字と斜体
これは//**太字と斜体**//のテキストです。 このテキストは**//本当に重要です**//。 This is //**bold and italicized**// text.
Plain Text
これは太字と斜体のテキストです。
このテキストは本当に重要です。
This is bold and italicized text.
下線
これは__下線付きの文字__です。 This is __underlined text__.
Plain Text
これは下線付きの文字です。
This is underlined text.
取り消し線
~~地球は平らだ。~~地球が丸いという事実はすべて知っている。 ~~The world is flat.~~ We now know that the world is round.
Plain Text
地球は平らだ。地球が丸いという事実はすべて知っている。
The world is flat. We now know that the world is round.
ハイライト
これは==ハイライトで強調されたテキスト==です。 ==地球が丸いという事実==はすべて知っている。 This is ==highlighted text==.
Plain Text
これはハイライトで強調されたテキストです。
地球が丸いという事実はすべて知っている。
This is highlighted text.
スタイルの組み合わせ
このように==//**__~~使用することも__~~**//==可能です。 **==__記号の順序==**__は重要ではありません。 ==他の書式同士で__交差設定==可能です。__
Plain Text
このように使用することも可能です。
記号の順序は重要ではありません。
他の書式同士で交差設定可能です。
❌  間違った使用例
テキスト スタイルは、記号の内側のすぐ隣にスペースを入れてはいけません。
This is ** bold text**. ❌ This is //italicized text //. ❌ Really //** very **// important text ❌
