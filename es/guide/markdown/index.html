Dar formato a texto
Este documento es la guía del usuario de Markdown de PenCake. El formato de texto es compatible desde PenCake 3.10.
PenCake usa Markdown para dar formato al texto. Markdown es un lenguaje en el que puede agregar elementos de formato al texto sin formato mediante símbolos. Markdown hace que formatear sea rápido y fácil.
En PenCake, para un formato más flexible e intuitivo, se ha modificado alguna sintaxis de Markdown de uso común. Por lo tanto, la siguiente sintaxis de Markdown solo es compatible con PenCake.
Estilos de texto
Negrita
Este es **un texto en negrita**. Macarrones**con**queso
Plain Text
Este es un texto en negrita.
Macarronesconqueso
Cursiva
Este es //el texto en cursiva//. Macarrones//con//queso
Plain Text
Este es el texto en cursiva.
Macarronesconqueso
Negrita y Cursiva
Este es un texto en //**negrita y cursiva**//. Este texto es **//muy importante**//. Macarrones**//con//**queso
Plain Text
Este es un texto en negrita y cursiva.
Este texto es muy importante.
Macarronesconqueso
Subrayado
Este es __un texto subrayado__. Macarrones__con__queso
Plain Text
Este es un texto subrayado.
Macarronesconqueso
Tachado
~~El mundo es plano.~~ Ahora sabemos que el mundo es redondo. Macarrones~~con~~queso
Plain Text
El mundo es plano. Ahora sabemos que el mundo es redondo.
Macarronesconqueso
Resaltado
Este es el texto ==resaltado==. Ahora sabemos que ==el mundo es redondo==. Macarrones==con==queso
Plain Text
Este es el texto resaltado.
Ahora sabemos que el mundo es redondo.
Macarronesconqueso
Combinación de estilos
Esto ==//**__~~también está__~~**//== disponible. El **==__orden de símbolos==**__ no importa. ==It is possible to __cross-specify== different styles__. ==Es posible especificar de __forma cruzada== diferentes estilos__.
Plain Text
Esto también está disponible.
El orden de símbolos no importa.
Es posible especificar de forma cruzada diferentes estilos.
❌  No hagas esto
Los estilos de texto no deben tener espacios justo al lado del interior de los símbolos.
Este es ** un texto en negrita**. ❌ Este es //el texto en cursiva //. ❌ Texto //** muy **// importante ❌
Plain Text
Encabezados

Encabezado nivel 1
