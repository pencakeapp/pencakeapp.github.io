Desktop-App

Die Desktop-App ist ein Dienst, der Premium-Nutzern zur Verfügung gestellt wird. (Kostenlose Nutzer können nur eingeschränkte Funktionen ausprobieren.)
💡
Die aktuelle Desktop-App ist eine Vorschauversion, nicht die endgültige Version, und einige Funktionen befinden sich noch in der Entwicklung. Mit fortschreitender Entwicklung werden fertiggestellte Funktionen schrittweise über automatische Updates bereitgestellt.


Unterschiede zur mobilen App

Schnellvorschau

Zeigen Sie beim Bearbeiten mit einer Tastenkombination sofort eine Vorschau Ihrer Formatierung an.

Suche im Eintrag

Finden Sie gewünschte Informationen ganz einfach direkt innerhalb eines Eintrags.
Auch eine Suchen-und-Ersetzen-Funktion steht im Bearbeitungsmodus zur Verfügung.

Verbesserte globale Suche

Durch eine Tastenkombination können Sie blitzschnell alle Inhalte in der App durchsuchen.
Springen Sie direkt zur exakten Fundstelle.

Exportfunktion

Exportieren Sie Ihre Inhalte als PDF- oder DOCX-Datei.

Benutzerhandbuch für die Desktop-App
Die Desktop-App von PenCake ist ein Service, der Premium-Nutzern zur Verfügung steht.
Kostenlose Nutzer unterliegen den folgenden Einschränkungen:

▪ Keine Möglichkeit, Einträge zu speichern
▪ Keine automatische Synchronisierung

Premium kann über die mobile Version erworben werden.
Nach dem Kauf richten Sie die Synchronisierung in folgender Reihenfolge ein: mobile Version → Desktop-Version, um Premium auf der Desktop-Version zu aktivieren.
Wenn Sie die Synchronisierung bereits in der mobilen Version verwenden, müssen Sie die Synchronisierung nur noch zusätzlich auf der Desktop-Version einrichten.

⚠️
Achtung: Falsche Synchronisierungseinstellungen können zu Datenverlust führen. Wenn Sie Hilfe benötigen, kontaktieren Sie uns bitte unter:
pencake.app@gmail.com
