Formatar texto
Este documento é o guia do usuário de Markdown do PenCake. A formatação de texto é suportada desde o PenCake 3.10.
PenCake usa Markdown para formatar texto. Markdown é uma linguagem que você pode adicionar elementos de formatação ao texto simples usando símbolos. O Markdown torna a formatação rápida e fácil.
No PenCake, para uma formatação mais flexível e intuitiva, algumas sintaxes Markdown comumente usadas foram modificadas. Portanto, a sintaxe Markdown abaixo é compatível apenas com PenCake.
Estilos de texto
Negrito
Este é **um texto em negrito**. Pizza**de**queijo
Plain Text
Este é um texto em negrito.
Pizzadequeijo
Itálico
Este é /um texto em itálico/. Pizza/de/queijo
Plain Text
Este é um texto em itálico.
Pizzadequeijo
Negrito e Itálico
Este é um texto em /**negrito e itálico**/. Este texto é **/muito importante**/. Pizza**/de/**queijo
Plain Text
Este é um texto em negrito e itálico.
Este texto é muito importante.
Pizzadequeijo
Sublinhado
Este é __um texto sublinhado__. Pizza__de__queijo
Plain Text
Este é um texto sublinhado.
Pizzadequeijo
Tachado
~~O mundo é plano.~~ Agora sabemos que o mundo é redondo. Pizza~~de~~queijo
Plain Text
O mundo é plano. Agora sabemos que o mundo é redondo.
Pizzadequeijo
Destaque
Este é o texto ==destacado==. Agora sabemos que ==o mundo é redondo==. Pizza==de==queijo
Plain Text
Este é o texto destacado.
Agora sabemos que o mundo é redondo.
Pizzadequeijo
Combinação de estilos
Isso ==//**__~~também está__~~**//== disponível. A **==__ordem dos símbolos==**__ não importa. ==É possível __cruzar especificar== estilos diferentes__.
Plain Text
Isso também está disponível.
A ordem dos símbolos não importa.
É possível cruzar especificar estilos diferentes.
❌  Não faça isso
Os estilos de texto não devem ter espaços logo ao lado do interior dos símbolos.
Este é um texto ** em negrito**. ❌ Este é um texto //em itálico //. ❌ Texto realmente //** muito **// importante ❌
